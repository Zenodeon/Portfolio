Cubey

Cubey – 3D Snake Game on a Physical Cube

Developed Cubey as part of a fast-paced 3-day challenge (aka. GameJam), with an additional 2–3 days of hardware preparation. The project features a fully custom hardware and software system: a cube equipped with 6 dot-matrix displays (one on each face), powered by an ESP32 and a 9-axis motion sensor.

Cubey runs a custom-built 3D Snake game designed to play across all six faces of the cube. The snake continuously moves “up,” and players guide its direction by physically rotating the cube. The gameplay is rendered and controlled through a Windows app built in Unity, which syncs real-time input from the hardware with the visual experience.

Key features:
- Custom hardware powered by ESP32 and 9-axis motion sensor
- Unique 3D Snake game spanning six dot-matrix displays
- Physical cube rotation as an intuitive control method
- Developed under a rapid 3-day challenge with 2–3 days dedicated to hardware setup
- Unity app for seamless gameplay visualization

Cubey is a demonstration of rapid prototyping, integrating physical motion sensing with spatial gameplay to create a novel interactive experience.
